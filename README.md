# 🚴 RideXpress - Bike Rental Management System

RideXpress is a simple and user-friendly **Bike Rental Management System** that allows users to rent bikes easily by selecting hours and riding modes.

This project is built using **HTML, CSS, JavaScript, and PHP**, with optional **MySQL database support** for user authentication.

---

## 📌 Features

* 🚴 6 Bikes Available for Rent
* ⏱ Enter Rental Hours
* ⚙ Choose Riding Mode (Normal / Premium / Luxury)
* 💰 Automatic Rent Calculation
* 👤 User Registration & Login System
* 🔐 Logout Functionality
* 🎨 Clean and Responsive UI

---

## 🚀 How to Run the Project

1. Install XAMPP (or any local server)
2. Start Apache and MySQL
3. Move project folder to:

   ```
   C:\xampp\htdocs\
   ```
4. Open browser and run:

   ```
   http://localhost/RideXpress/register.html
   ```

---

## 💡 How It Works

1. User registers an account
2. Logs into the system
3. Views available bikes
4. Enters rental hours and selects mode
5. System calculates total rent instantly
